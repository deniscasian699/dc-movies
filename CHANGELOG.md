# Changelog

All notable changes to DC Movies are documented here.

---

## Version 2.0.0 — 2026

### Major Release

#### Added

- **Live TV** — Over 9,000 channels from more than 150 countries
- **Multi-language** — Full Romanian translation support
- **License System** — Gumroad integration for subscriptions
- **Cross-device Sync** — Real-time progress sync via Firebase
- **PWA Support** — Install as native app on any device
- **Keyboard Shortcuts** — Full keyboard navigation for player
- **Actor Pages** — Browse by actor with complete filmographies
- **Anime Section** — Dedicated anime browsing experience

#### Changed

- Complete UI redesign with new color scheme
- Improved video player with multiple server fallback
- Better mobile experience with touch optimizations
- Faster page loads with intelligent caching

#### Fixed

- Episode progress not saving correctly for TV shows
- Search not finding some content
- Mobile fullscreen issues on iOS
- Various UI bugs and glitches

---

## Version 1.5.0 — 2026

#### Added

- Watch history tracking
- Favorites and Watchlist functionality
- User ratings from 1 to 10
- Continue watching section on homepage

#### Changed

- Improved search with autocomplete suggestions
- Better error handling throughout the app

---

## Version 1.0.0 — 2026

### Initial Release

- Basic movie and TV browsing
- Video playback functionality
- User authentication
- Responsive design for all devices

---

## Version Format

This project uses Semantic Versioning:

- **MAJOR** — Breaking changes
- **MINOR** — New features that are backwards compatible
- **PATCH** — Bug fixes
