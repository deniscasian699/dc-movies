## 📖 About

**DC Movies** is a personal streaming platform built as a hobby and educational project. It provides a modern, Netflix-like experience for watching movies, TV shows, and live TV channels.

### Why This Project?

- 🎓 **Educational** — Built to learn modern web development techniques
- 🛠️ **Technical Challenge** — Real-time sync, PWA implementation, video streaming
- 🎨 **Design Practice** — Creating a polished, professional user interface

### Important Notes

| ⚠️ | This website is **not indexed** on search engines due to potential copyright concerns |
|---|---|
| 🔒 | This is a **private/personal** project, not a commercial service |
| 💰 | Subscription fees cover server costs and API usage only |

## ✨ Features

### 🎥 Content Library

- **Movies** — Browse trending, popular, top-rated, and upcoming films
- **TV Shows** — Full series support with season and episode navigation
- **Live TV** — Over 9,000 channels from more than 150 countries
- **Anime** — Dedicated anime section for Japanese animation
- **Actors** — Browse by actor and view complete filmographies

### 👤 User Features

- **Watch Progress** — Resume exactly where you left off on any device
- **Cross-Device Sync** — Your data syncs across all devices in real-time
- **Favorites** — Save movies and shows you love for quick access
- **Watchlist** — Queue content to watch later
- **History** — Track everything you have watched
- **Ratings** — Rate content from 1 to 10 with your personal score

### 🎮 Player Features

- **Multiple Servers** — Automatic fallback if one server fails
- **Resume Playback** — Continue from your last position with one click
- **Keyboard Shortcuts** — Full keyboard control including Space, F, M, N, and more
- **Picture-in-Picture** — Watch while browsing other content
- **Chromecast** — Cast to your TV wirelessly
- **Quality Selection** — Choose your preferred streaming quality

### 📱 Platform Support

- **Progressive Web App** — Install as a native app on any device
- **Responsive Design** — Perfect experience on mobile, tablet, and desktop
- **Offline Mode** — Basic functionality available without internet
- **Multi-language** — Full support for English and Romanian

## 🛠️ Tech Stack

| Category | Technology | Purpose |
|----------|------------|---------|
| Frontend | Vanilla JavaScript ES6+ | No frameworks, pure JavaScript modules |
| Styling | CSS3 with Custom Properties | Responsive and themeable design |
| Authentication | Firebase Auth | Google and Email/Password login |
| Database | Cloud Firestore | Real-time user data synchronization |
| API Proxy | Cloudflare Workers | TMDB API proxy with caching |
| Hosting | Cloudflare Pages | Fast global CDN delivery |
| Payments | Gumroad | License and subscription management |
| Video | HLS.js | Adaptive bitrate streaming |
| Live TV | IPTV-ORG API | Over 9,000 free channels |
| Content Data | TMDB API | Movies and TV metadata |
